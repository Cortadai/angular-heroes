Este repositorio contiene una **aplicación web de gestión de superhéroes** construida con Angular 14 y Angular Material. Es una aplicación CRUD completa que permite a los usuarios autenticarse (con login estándar o anónimo) y luego administrar una base de datos de superhéroes de DC Comics y Marvel. Las funcionalidades principales incluyen: listar todos los héroes en una cuadrícula responsive, ver detalles individuales de cada héroe, agregar nuevos superhéroes, editar los existentes, eliminar héroes con diálogo de confirmación, y buscar héroes mediante un sistema de autocompletado en tiempo real. La aplicación utiliza guards para proteger las rutas del módulo de héroes, se comunica con un backend REST API corriendo en localhost:3000 (probablemente JSON Server), maneja imágenes de forma inteligente con pipes personalizados, y demuestra buenas prácticas de Angular como lazy loading, arquitectura modular, inyección de dependencias, y un diseño Material Design moderno y responsivo.
